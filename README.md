# ⚛️ Trabalho de Conclusão — React + Vite

Projeto de conclusão desenvolvido com **React**, **Vite**, **React Router DOM** e **Axios**, com navegação entre páginas e consumo de API externa.

---

## 🚀 Tecnologias utilizadas

- [React 19](https://react.dev/) — biblioteca para construção da interface
- [Vite 8](https://vite.dev/) — ferramenta de build e servidor de desenvolvimento
- [React Router DOM 7](https://reactrouter.com/) — navegação entre páginas (SPA)
- [Axios](https://axios-http.com/) — requisições HTTP para consumo de API
- CSS — estilização dos componentes

## ⚙️ Como rodar o projeto

**Pré-requisitos:** Node.js instalado na máquina.

```bash
git clone https://github.com/NezyMZ/Trabalho-Conclusao-React-Vite.git
cd Trabalho-Conclusao-React-Vite
npm install
npm run dev
```

Acesse no navegador: `http://localhost:5173`

---

## 📦 Scripts disponíveis

| Comando | Descrição |
|---|---|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera a versão de produção na pasta `dist/` |
| `npm run preview` | Pré-visualiza o build de produção localmente |
